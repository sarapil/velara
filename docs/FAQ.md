# Velara — Frequently Asked Questions
# فيلارا — الأسئلة الشائعة

## ❓ How do I add a property?

Go to Velara > Properties > New Property. Add units, facilities, and amenities.

## ❓ How does rent collection work?

Leases generate rent schedules. Invoices are auto-created and tracked.

## ❓ Can I manage multiple properties?

Yes — unlimited properties with portfolio-level dashboards.
