# Responsive Matrix — مصفوفة الاستجابة

## Breakpoints

| Breakpoint | Width | Layout |
|-----------|-------|--------|
| Mobile | 320px – 767px | Single column, bottom nav |
| Tablet | 768px – 1023px | Two column, collapsible sidebar |
| Desktop | 1024px – 1439px | Full layout with sidebar |
| Large | 1440px+ | Extended layout with panels |

## Per-Screen Behavior

| Screen | Mobile | Tablet | Desktop | Large |
|--------|--------|--------|---------|-------|
| Dashboard | ✅ Stack | ✅ 2-col | ✅ Full | ✅ Extended |
| List Views | ✅ Cards | ✅ Table | ✅ Table | ✅ Table |
| Forms | ✅ Single | ✅ Single | ✅ Sections | ✅ Sections |
