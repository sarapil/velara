## يعمل مع ERPNext الحالي لديك

Velara لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/velara
bench --site your-site install-app velara
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
