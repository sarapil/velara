## Key USPs (Unique Selling Propositions)
1. **Arabic-first design** — not a translation, built for Arabic workflows
2. **CAPS capability system** — most granular permission model in the market
3. **Visual components** — frappe_visual for stunning dashboards
4. **AI integration** — multi-model AI router with per-app features
5. **ERPNext ecosystem** — seamless integration with accounting, HR, inventory

## Target Segments
- **SMEs (10-100 employees)**: Community tier, self-service
- **Mid-market (100-500)**: Professional tier, guided onboarding
- **Enterprise (500+)**: Enterprise tier, dedicated support
